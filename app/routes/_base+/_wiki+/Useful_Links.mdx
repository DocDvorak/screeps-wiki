## Private Server Launchers

### Custom Launchers

These are the community preferred solutions for hosting a private server.

- [Jomik: Screeps Server](https://github.com/Jomik/screeps-server) - Easy to use, Docker-based.
- [Screeps Launcher](https://github.com/screepers/screeps-launcher) - Supports both Docker and direct setups.

### Official Launcher

- **GUI Launcher**: This launcher comes with the standard version of Screeps World on Steam. It provides a GUI for managing the server, viewing server logs, and a CLI interface to control the server.
- **Console Launcher**: Allows launching the server from a command line without the GUI. Prerequisites include Node 10, Python 2, and developer build tools

### Screeps Clients

Steamless client runs your copy of Screeps in the browser and works with private servers on macOS, Linux and Windows.

- [screepers-steamless-client](https://github.com/screepers/steamless-client) - Screepers version is updated with additional features.
- [screeps-steamless-client](https://github.com/laverdet/screeps-steamless-client) - Original Screeps steamless client.
- [xxscreeps](https://github.com/laverdet/xxscreeps) - Optimized rewrite of the Screeps client.
- [Screeps3D](https://github.com/thmsndk/Screeps3D) - 3D client for Screeps.

## Screeps Mods

These mods are commonly installed on Screeps Private Servers.

- [screepsmod-mongo](https://github.com/screepsmods/screepsmod-mongo) - Recommended - Private servers run much better with MongoDB compared to the default LokiJS db.
- [screepsmod-auth](https://github.com/ScreepsMods/screepsmod-auth) - Recommended - Adds basic auth and allows players to upload scripts via API
- [screepsmod-admin-utils](https://github.com/ScreepsMods/screepsmod-admin-utils) - Useful commands for server admins.
- [screepsmod-market](https://github.com/ScreepsMods/screepsmod-market)
- [screepsmod-history](https://github.com/ScreepsMods/screepsmod-history) - Warning: may corrupt server after 1M ticks.
- [screepsmod-speedrun](https://github.com/admon84/screepsmod-speedrun) - Removes bots and disables invaders for a speedrun.
- [ScreepsMods](https://github.com/orgs/ScreepsMods/repositories)
